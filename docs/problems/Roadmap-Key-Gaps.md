Use this as the working marketing-risk matrix for ContentCraft. Keep phrasing concise so it is AI- and roadmap-friendly.

## Color key

- **Green** – defensible now
- **Yellow** – partially true; wording is ahead of the product
- **Red** – high trust risk; soften the claim or build the feature fast
